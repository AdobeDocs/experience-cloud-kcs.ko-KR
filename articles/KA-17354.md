# 상승도 값이 Target UI에서 사라졌습니다.

## 설명 {#description}

<b>환경</b>
- Target



<b>문제/증상</b>
상승도 값은 Target UI에 표시되었지만, 이제 동일한 날짜 범위를 확인할 때 사용할 수 없습니다.


## 해상도 {#resolution}




<b>팩트 :</b> UI에 신뢰도 0인 경험에 대한 상승도가 표시되지 않습니다



<b>설명 : </b>UI에 상승도가 표시되면 신뢰도의 값은 0입니다. 그러나 방문자가 더 많은 낮 동안 신뢰도는 0으로 감소했습니다

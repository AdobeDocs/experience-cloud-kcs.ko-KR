# 네트워크 오류로 인해 프로젝트를 저장하지 못했습니다.

## 설명

프로젝트를 저장하려고 할 때 간헐적으로 다음 오류 메시지가 표시됩니다.
<br> 
<br>&quot;현재 프로젝트를 저장할 수 없습니다. 페이지를 새로 고친 후 다시 시도할 수 있습니다.&quot;<br><br>![](assets/___89bd255f-c855-ed11-bba3-6045bd0065f9___.png)

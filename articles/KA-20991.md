# 분석 데이터 피드의 geo_country 열에 있는 &#39;\*\*\*&#39;

## 설명 {#description}

<b>환경</b><br>분석<br> <br><b>문제/증상</b><br>에 대한 이유 `<b>*` 의 값 `geo_country` 분석 데이터 피드 파일의 열입니다.



 

## 해상도 {#resolution}

`</b>*` 히트에 `IPv6` 테스트를 위해 지정된 값입니다. 이러한 경우 `geo_country` 반환 `<b>*` 및 `geo_city` 예약된 을 반환합니다.

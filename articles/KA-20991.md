# Analytics 데이터 피드의 geo_country 열에 있는 &#39;\*\*&#39;

## 설명 {#description}

분석 데이터 피드 파일의 geo_country 열에 있는 \*\*\* 값에 대한 설명입니다.



<br> 

## 해결 방법 {#resolution}

<b>\*\*\&lt;/b>* 히트에 테스트를 위해 지정된 IPv6 값이 있을 때 발생합니다. 이러한 경우 geo_country 는 <b>\*\*\&lt;/b>* 및 geo_city 반환 <b>예약</b>.

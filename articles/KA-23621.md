# 고정 날짜를 선택한 후에도 Report Builder의 캘린더 로드 중

`[` 문제`]` <br>
선택한 후에도 Report Builder의 캘린더가 로드될 수 있습니다. `[` 고정 날짜`]` .

## 설명 {#description}


### `[` 환경`]`

- Adobe Analytics(Report Builder)



## 해결 방법 {#resolution}

`[` 해결 방법`]` <br><br>
일부 환경에서 캘린더를 로드하지 못할 수 있습니다.
로드 캘린더 를 찾을 수 있는 경우:

1. 팝업 메뉴 표시를 위한 오른쪽 버튼 클릭
2. 클릭 `[` 새로 고침`]`

   \*\* 또는 누르기 `[` F5`]`  페이지 다시 로드 키

# Insights 구성 설정 시 내부 서버 오류 발생

## 설명 {#description}

<b>환경</b>
Experience Manager 6.5


<b>문제/증상</b>
인사이트 구성을 구성할 때 내부 서버 오류를 가져옵니다.

<b>재현 단계:</b>

1. 이동 <b>도구</b> - <b>자산</b> - <b>통찰력 구성</b>.
2. Enter 키 *통찰력 구성* analytics 세부 사항.
3. 클릭 <b>인증</b>.
4. 메시지에 오류 팝업 *내부 서버 오류* 이 나타납니다.



## 해상도 {#resolution}


1. 자산 구성 *통찰력 구성* 를 입력합니다.
2. 경로의 패키지 만들기 `/etc/cloudservices/sitecatalyst/assetinsights/`.
3. 더 높은 인스턴스에 패키지를 설치합니다.


<b>참고</b>: 이는 AMS 또는 on-prem 인스턴스에만 적용할 수 있습니다.

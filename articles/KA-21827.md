# Adobe Target 피드 파일 가져오기 기본 정보

## 설명 {#description}

권장 사항 피드를 가져오는 중입니다. 업로드할 때마다 모든 값을 덮어쓰시겠습니까? 또는 덮어쓴 수정된 값 간의 차이만 있습니까?

## 해결 방법 {#resolution}


권장 사항 피드의 경우 모든 값이 첫 번째 업로드 시에만 업데이트됩니다. 이후에 업로드하면 변경된 값만 업데이트됩니다. 즉, 새 값만 기존 값과 비교하여 업데이트됩니다.

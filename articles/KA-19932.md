# AEP - SFTP 데이터 소스 커넥터를 사용하는 동안 SFTP에 추가할 파일의 최대 크기는 얼마입니까?

## 설명

SFTP 데이터 소스 커넥터를 사용하는 동안 SFTP에 추가할 파일의 최대 크기는 얼마입니까?

## 해상도


현재 최대 1500개의 파일이 있으며 모두 최대 100GB 크기(누적)에 해당합니다.

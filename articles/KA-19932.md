# SFTP에 넣을 파일의 최대 크기는 얼마입니까 | AEP

## 설명 {#description}

<b>환경</b>
Experience Platform


<b>문제/증상</b>
SFTP(SSH File Transfer Protocol) 데이터 소스 커넥터를 사용하는 동안 SFTP에 추가할 파일의 최대 크기는 얼마입니까?


## 해상도 {#resolution}

현재 최대 1500개의 파일이 있으며 모두 최대 100GB 크기(누적)에 해당합니다.

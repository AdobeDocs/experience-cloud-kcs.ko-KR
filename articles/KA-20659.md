# 분류 규칙 빌더의 전환 확인 기간이 1개월로 되돌아갑니다.

## 설명 {#description}

<b>환경</b>
- Adobe Analytics

<b>문제/증상</b><br>분류 규칙 빌더의 전환 확인 기간은 예를 들어 6개월로 설정한 후 1개월로 되돌아갑니다.
<br> 

## 해결 방법 {#resolution}


전환 확인 기간은 규칙 세트에 설정된 임시 옵션입니다. 규칙 세트가 처리된 후 1개월로 되돌아갑니다.

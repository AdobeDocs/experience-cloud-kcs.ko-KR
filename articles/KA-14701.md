# Adobe Campaign: 오류: &#39; 식을 구문 분석할 수 없습니다.[@registrationToken]`

## 설명

모바일 앱 채널을 사용하려고 할 때 다음 오류가 발생했다고 보고되었습니다.

```
error XTK-170036 Unable to parse expression '@registrationToken'
```

## 해상도


해결 방법은 [!UICONTROL 모바일 앱 채널] 를 반환합니다.

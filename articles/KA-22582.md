# New Relic에 프로덕션 인스턴스가 없음

## 설명 {#description}


### <b>환경</b>

Experience Manager



### <b>문제/증상</b>

프로덕션 인스턴스가 응용 프로그램 모니터링 서비스에 나타나지 않습니다. *New Relic*.


## 해결 방법 {#resolution}


프로덕션 환경이 New Relic에 나타나게 하려면 프로덕션 파이프라인을 완료해야 합니다.

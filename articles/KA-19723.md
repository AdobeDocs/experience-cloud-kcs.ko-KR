# AEM 6.1에서 6.5로 콘텐츠를 마이그레이션할 수 없음


Adobe Experience Manager v6.1에서 v6.5로 콘텐츠를 마이그레이션하는 동안 스택 추적이 발생하면 먼저 AEM v6.3으로 마이그레이션한 다음 v6.5로 진행하는 것이 좋습니다.

## 설명 {#description}


<b>환경</b>

Adobe Experience Manager 6.5

<b>문제/증상</b>

다음 스택 추적은 AEM v6.1에서 v6.5로 콘텐츠를 마이그레이션하는 동안 발생합니다.

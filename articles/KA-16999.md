## 설명

<b>환경</b>
- Customer Journey Analytics
- Analytics



<b>문제/증상</b>
분류(SAINT) API에서 분류 API 가져오기 오류 &quot;Relation 0이 현재 데이터를 가져오도록 설정되어 있지 않습니다.&quot;에 대한 이유는 무엇입니까?

# 오류: 사용자 지정 콘솔의 org.apache.sling.api.request.TooManyCallsException


이 문서에서는 AEMaaCS 문제를 해결하는 방법에 대해 설명합니다. *org.apache.sling.api.request.TooManyCallsException* 사용자 지정 콘솔에 오류가 표시됩니다. 경로를 오버레이할 수 있습니다 —` …/aem/apps/lv/console/lvjobs/content/lvjobs.html` 또는 속성을 사용하여 전역적으로 제한을 설정합니다. `'sling.max.record.requests'` OSGi 구성 PID에서 `org.apache.sling.engine.impl.SlingMainServlet`.

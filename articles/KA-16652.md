# 외부 서버에 대한 클라이언트측 인증서 인증 | Adobe Campaign

## 설명


일부 데이터를 가져오기 위해 Adobe Campaign에서 외부 시스템으로 REST API를 호출하기 위해 SSL 인증서를 가져올 수 있습니까? 외부 응용 프로그램은 인증서 기반 인증을 지원합니다.


## 해상도


Adobe 캠페인이 외부 서버에 대한 클라이언트측 인증서 인증을 지원하지 않습니다.

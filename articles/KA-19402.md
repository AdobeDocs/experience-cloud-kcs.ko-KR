# 이메일 디자이너에 표시되지 않는 조각 목록

## 설명 {#description}

<b>환경</b>
- Campaign Standard



<b>문제/증상</b><br>문제 설명 - 새 이메일 디자이너를 사용할 때 조각 목록에 OOTB 조각만 나타나고 목록을 새로 고치려 하면 목록이 영구적으로 로드됩니다.

## 해상도 {#resolution}


이 문제는 기존 편집기를 사용하여 조각을 만들 때 발생할 수 있습니다. 조각을 열고 새 편집기를 통해 조각을 저장하면 조각이 업데이트되고 나중에 목록이 작동합니다.

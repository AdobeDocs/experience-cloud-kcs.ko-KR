# [!UICONTROL 파일 수집기] acs의 활동 대체 요소

## 설명

이 없으므로 [!UICONTROL 파일 수집기] acs에서는 다음 활동을 사용하여 만들 수 있으며, 같은 기능을 사용할 수도 있습니다

## 해상도


![](assets/8e624e34-2cce-ec11-a7b5-0022480a8e40.png)



다음 [!UICONTROL 파일 수집기] 에서는 외부 서버에서 파일 이름 패턴을 기반으로 여러 파일을 다운로드할 수 있습니다. 각 파일은 [!UICONTROL 파일 로드] 활동. 업데이트로 데이터를 제공합니다.
완료되면 다음 파일 등으로 계속 진행합니다.
파일 전송에는 [!UICONTROL 내역 설정]에 대해 알고 있듯이 [!UICONTROL 파일 수집기].

# Adobe Campaign: API 504 시간 초과 오류

## 설명

<br>문제/증상<br><br>
- Campaign에 API를 호출할 때 504 오류 수신



환경
- Apache RequestReadTimeout 옵션이 20초(기본 설정)로 설정되어 있습니다



## 해상도

해결/해결 단계
- 일부 API 호출이 기본 20초 및 시간 초과를 초과할 수 있으므로 Amazon ELB(60초)와 일치하도록 이 시간 제한을 늘립니다.

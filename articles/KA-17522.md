# 단계별 Adobe I/O: 인증 설정 방법

## 설명

<br>이 문서에서는 실행할 수 있는 인증서, 키 및 토큰을 생성하는 방법을 소개합니다.<br><br><br><br><br>목표<br><br><br><br><br><br>
명확하고 재현 가능한 단계에 따라 [!DNL Target] 등 DMA 솔루션에 대한 API 호출을 실행할 수 있도록 Adobe I/O 전달자 토큰을 생성하기 위함입니다.


## 해상도

<br><br>단계<br><br><br><br> <br><br>
<b>Adobe I/O API 호출을 실행할 수 있는 전달자 토큰을 생성하는 단계:</b>

1. 공개 키와 공개 인증서 생성
2. Adobe I/O 콘솔 내에서 통합 만들기
3. JWT 토큰 생성
4. JWT를 Access Bearer 토큰으로 교환
5. API 호출을 실행할 사용자 Access Bearer 토큰


- <b>다음 문서에 따라 공개 키 및 공개 인증서 생성:</b> https://www.adobe.io/apis/cloudplatform/console/authentication/createcert.html


   Mac에서 다음 명령이 터미널에 기본 제공됩니다. 

   컴퓨터에서 [!DNL Cygwin](또는 개인 환경 설정에 적합한 도구)을 다운로드하고 명령줄에서 실행해야 합니다. 다음은 [!DNL Cygwin]을 다운로드 및 설치하는 단계입니다.

   1. https://cygwin.com/install.html 검색
   2. `setup-x86_64.exe` 다운로드 및 실행

   <b>*참고</b>: 홈 디렉터리는 다음과 같습니다.`C:\cygwin64\home\WINDOWSUSER`

   설치 단계에서 추가 패키지를 검색하고 설치할 수 있습니다. “`curl`” 및 “`ssh.`”와 관련된 모든 코드를 설치하는 것이 좋습니다.*


   실행할 명령은 다음과 같습니다.
   <br> <br> <br>

   | `$ openssl req -nodes -text -x509 -newkey rsa:2048 -keyout secret.pem -out certificate.pem -days 356` |
   | --- |

<br> 

    참고&lt;b>: 인증서 생성을 위해 몇 가지 질문을 작성하는 경우 &lt;/b>아래 스크린샷
을 참조하십시오.![rtaImage](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_0/step_par/image/rtaImage.png "rtaimage")<br> 

    파일이 생성되면 다음 명령을 사용하여 “secret.key”를 “secret.pem”`으로 변환합니다.
<br> <br> <br>

    | `$ openssl pkcs8 -topk8 -inform PEM -outform DER -` `in` `secret.pem  -nocrypt  secret.key` |

   - “생성된 JWT” 아래에 있는 “복사” 아이콘을 클릭합니다.

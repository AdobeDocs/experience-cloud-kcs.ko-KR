# Adobe I/O 설정 방법: 인증 - 단계별

## 설명

<br>이 문서에서는 실행할 수 있도록 인증서, 키 및 토큰을 만드는 방법을 안내합니다.<br><br><br><br><br>목표<br><br><br><br> <br><br>
와 같은 DMA 솔루션에 대한 API 호출을 실행하기 위해 Adobe I/O 베어러 토큰을 생성하는 명확하고 재현가능한 단계를 제공합니다. [!DNL Target].


## 해상도

<br><br>단계<br><br><br><br> <br><br>
<b>Adobe I/O API 호출을 실행할 베어러 토큰을 만드는 절차:</b>

1. 개인 키 및 공개 인증서 생성
2. Adobe I/O 콘솔 내에서 통합 만들기
3. JWT 토큰 생성
4. 액세스 베어러 토큰에 대한 Exchange JWT 토큰
5. API 호출을 실행할 사용자 액세스 베어러 토큰


- <b>다음 설명서에 따라 개인 키 및 공개 인증서를 생성합니다.</b> https://www.adobe.io/apis/cloudplatform/console/authentication/createcert.html


   Mac에서는 터미널에 다음 명령이 빌드되어 있습니다. 

   컴퓨터에서 다운로드 받아야 합니다 [!DNL Cygwin] (또는 개인 기본 설정의 다른 도구)를 사용하여 명령줄에서 실행합니다. 다운로드 및 설치 단계는 다음과 같습니다 [!DNL Cygwin]:

   1. https://cygwin.com/install.html 로 이동합니다.
   2. 다운로드 및 실행 `setup-x86_64.exe`

   <b>*참고</b>:  홈 디렉토리는 다음과 같습니다. `C:\cygwin64\home\WINDOWSUSER`

   설치 단계에서 추가 패키지를 검색하고 설치할 수 있습니다. &quot;&quot;과(와) 관련된 모든 것을 설치하는 것이 좋습니다`curl`&quot; 및 &quot;`ssh.`&quot;*


   다음은 실행할 명령입니다.
   <br> <br> <br>

   | `$ openssl req -nodes -text -x509 -newkey rsa:2048 -keyout secret.pem -out certificate.pem -days 356` |
   | --- |

<br> 

    &lt;b>참고&lt;/b>:  인증서를 생성하기 위해 입력해야 하는 몇 가지 질문이 있는 경우 아래 스크린샷을 참조하십시오.
![rtaImage](https://helpx.adobe.com/content/dam/help/en/marketing-cloud-core/kb/adobe-io-authentication-step-by-step/jcr%3acontent/main-pars/procedure/proc_par/step_0/step_par/image/rtaImage.png "rtaImage")<br> 

    파일이 만들어지면 다음 명령을 사용하여 &#39;secret.key&#39;를 &#39;secret.pem&#39;으로 변환합니다.
<br> <br> <br>

    | &#39;$ openssl pkcs8 -topk8 -inform PEM -outform DER -&#39; &#39;in&#39; &#39;secret.pem -nocrypt secret.key&#39; |

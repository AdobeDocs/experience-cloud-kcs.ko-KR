# 자산 게시가 &quot;나중에&quot; 상태에서 중단됨

## 설명 {#description}


<b>환경</b>

Adobe Experience Manager

<b>문제/증상</b>

한 고객이 여러 자산, 특히 이미지를 게시하는 동안 지속적인 오류가 발생했습니다. 종종 이미지가 종료되지 않는 상태 &quot;나중에&quot;로 유지되었지만 다른 경우에는 &quot;게시되지 않음&quot;으로 실패하면서 &quot;보류 중&quot; 상태로 유지되었습니다.

## 해결 방법 {#resolution}


에서 볼 수 있듯이 `error.log`, 문제는 다음으로 인해 발생합니다. `"javax.jcr.ValueFormatException: cq:tags = value is single-valued"`

다음 `cq:tags` 속성이 실제로 ``String Array``: 다중 값 속성입니다. 그러나 이 경우 유형이 다음과 같습니다 ``String`` 오류가 발생했습니다.

를 변경하면 `cq:tags` 속성 유형 대상 ``String Array``, 자산이 성공적으로 게시되었습니다.

<b>설명서 링크</b>

에 대한 추가 도움말 `cq:tags` 속성은 다음 설명서 링크에서 찾을 수 있습니다.
[https://experienceleague.adobe.com/docs/experience-manager-65/developing/platform/tagging/framework.html?lang=en](https://experienceleague.adobe.com/docs/experience-manager-65/developing/platform/tagging/framework.html?lang=en)

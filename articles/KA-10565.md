<b>환경</b>
AEM 6.2 및 Brand Portal

<b>원인</b>
원인은 AEM 인스턴스와 Brand Portal 클라우드 간의 시간 차이입니다.
이렇게 하면 JWT가 만료됩니다.

<b>지정된 메시지</b>


```
Failed to get an access token 
JWT has expired
```



## 해상도


<b>해상도</b>

1. AEM의 OS 시간을 확인합니다.
2. 시간을 유효한 시간으로 변경하거나 ntp를 사용합니다.

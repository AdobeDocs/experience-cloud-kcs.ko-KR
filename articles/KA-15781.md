# Campaign Classic - 설명 분석에서 채워지지 않은 표

## 설명


설명 분석 보고서를 내보낼 때 차트 부품에 의해 그래프가 내보내지지 않습니다.


## 해상도


Excel 옵션을 사용하고 차트 섹션을 올바르게 내보내는 경우 보고서가 올바르게 추출되므로 엔지니어링 팀이 이를 개선 요청으로 수락했습니다.

이 문제는 21.1.4 릴리스(빌드 9343) 이상에서 수정되었습니다.

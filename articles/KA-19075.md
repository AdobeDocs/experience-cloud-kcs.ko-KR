# AAM 데이터 공유에 AEP 활성화

## 설명

AEP에서 AAM으로 두 개 이상의 샌드박스에 대한 피로도 통합을 활성화하는 방법

## 해상도


- 세그먼트 통합에 대해 샌드박스를 하나만 구성할 수 있습니다.
- 샌드박스를 활성화하려면 AAM 커넥터가 다른 샌드박스에 없는지 확인합니다.

# AAM 데이터 공유에 AEP 활성화

## 설명

활성화 방법 [!DNL Segue] AEP에서 AAM으로 두 개 이상의 샌드박스를 통합하는 방식

## 해상도


- 샌드박스는 하나만 구성할 수 있습니다 [!DNL Segue] 통합.
- 샌드박스를 활성화하려면 AAM 커넥터가 다른 샌드박스에 없는지 확인합니다.
